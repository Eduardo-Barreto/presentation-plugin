Workflow

Como Funciona

Um fluxo simples de 4 passos para criar apresentacoes profissionais.

01

Descreva o conteudo

Escreva um prompt detalhando topicos, estilo visual e estrutura desejada.

02

Claude gera o HTML

O modelo cria um arquivo HTML completo com CSS e JavaScript inline.

03

Revise e ajuste

Peca modificacoes especificas: cores, animacoes, novos slides.

04

Apresente

Abra no navegador e use fullscreen. Funciona offline.

Exemplo

Um Prompt Eficaz

Estruture seu pedido com tema, estilo e requisitos tecnicos.

$ claude
Crie uma apresentacao HTML single-file sobre [TEMA].

ESTILO:
- Dark mode minimalista
- Tipografia: Geist
- Cores: monocromatico com acentos azul/verde

ESTRUTURA:
- 8 slides com scroll-snap
- Hero, problema, solucao, features, demo, CTA

TECNICO:
- Animacoes com Intersection Observer
- Navegacao por teclado
- Responsivo
